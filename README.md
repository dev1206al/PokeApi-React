## Probar el proyecto

Una vez levantado el proyecto, puedes probarlo siguiendo estos pasos:

1. En tu navegador, abre la dirección `http://localhost:3000/`.
2. La página mostrará un listado de Pokémon.
3. Utiliza la barra de búsqueda para encontrar un Pokémon por su nombre.
4. Haz clic en un Pokémon para ver detalles adicionales.

¡Listo! Ahora puedes explorar y utilizar la aplicación PokeApi-React localmente.
